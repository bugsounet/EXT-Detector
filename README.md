# EXT-Detector

## Plugin for `MMM-GoogleAssistant`

Keywords Detectors for MMM-GoogleAssistant !

Multi-keyword listener with
* Snowboy Features
* Porcupine Features

It will include:
  * Ok Google
  * Hey Google
  * Jarvis
  * Hey Siri
  * Computer
  * ...

All informations is now the [wiki](http://wiki.bugsounet.fr/en/EXT-Detector)
